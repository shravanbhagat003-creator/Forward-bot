# Telegram CC Bot

A Telegram bot that monitors channels for credit card information and forwards them with specific formatting.

## Deployment on Railway

1. Fork this repository
2. Connect to Railway
3. Add environment variables
4. Deploy!

## Environment Variables

- `API_ID`: Your Telegram API ID
- `API_HASH`: Your Telegram API Hash
- `STRING_SESSION`: Your Telegram session string
- `SOURCE_CHANNEL`: Source channel ID
- `TARGET_CHANNEL`: Target channel ID
